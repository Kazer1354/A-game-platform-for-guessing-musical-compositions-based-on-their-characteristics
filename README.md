# 🎵 MusicGuess — Игровая платформа для угадывания музыкальных произведений

**MusicGuess** — это веб-приложение, основанное на механике популярной игры Wordle, но адаптированное для мира музыки. Пользователи должны угадать загаданную песню, анализируя подсказки по её объективным характеристикам: год выпуска, длительность, темп, тональность, жанр, страна исполнителя и популярность.

---

## 🎮 Как играть

1. **Откройте сайт** `https://musicguess.app` (или локально `http://localhost:3000`).
2. **Игра начинается автоматически** — система загадывает случайную песню из базы данных.
3. **Введите название песни** в строку поиска. Система предложит выпадающий список совпадений (название + исполнитель).
4. **Выберите трек** из списка, чтобы подтвердить свою догадку.
5. **Проанализируйте подсказки** — для каждой характеристики отображается результат сравнения с загаданной песней.
6. **Продолжайте угадывать**, пока не найдёте песню или не закончатся попытки.

---

## 🎨 Цветовые подсказки и индикация

Для каждой попытки система сравнивает характеристики введённой песни с загаданной и отображает результат:

| Индикатор | Значение | Пример |
|:---------:|----------|--------|
| 🟢 **Зелёный** | Полное совпадение | Жанр «Рок» = загадан «Рок» |
| 🟡 **Жёлтый** | Частичное совпадение | Страна «Франция» → континент «Европа» совпадает |
| ⬜ **Серый** | Несовпадение | Страна «Япония» → континенты разные |
| ⬆️ **Стрелка вверх** | Значение больше | Длительность 240 сек → загадано 180 сек |
| ⬇️ **Стрелка вниз** | Значение меньше | Год 2015 → загадан 2019 |

### Сравниваемые характеристики:
- 📅 **Год выпуска** (стрелки + жёлтый при разнице ≤ 3 лет)
- ⏱️ **Длительность** (стрелки + жёлтый при разнице ≤ 30 сек)
- 🎵 **Темп (BPM)** (стрелки + жёлтый при разнице ≤ 10%)
- 🎼 **Тональность** (зелёный при совпадении)
- 🎸 **Жанр** (зелёный при совпадении)
- 🌍 **Страна исполнителя** (зелёный при совпадении, жёлтый при совпадении континента)
- 📊 **Популярность** (стрелки с числовым значением)

## 🕹️ Режимы игры

Перед началом игры можно выбрать фильтры для усложнения или упрощения:

- **Обычный режим** — загадывается любая песня из базы
- **По стране** — только исполнители из выбранной страны (например, Россия, США, Великобритания)
- **По жанру** — только треки определённого жанра (рок, поп, хип-хоп, электроника)
- **По десятилетию** — песни выбранного десятилетия (1980-е, 2000-е и т.д.)
